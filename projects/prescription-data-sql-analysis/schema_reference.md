# Schema Reference

Expected table: data

Columns:
- triage_time (timestamp)
- department (text)
- shift (text)
- priority (text)
- prescriptions_count (integer)
- triage_time_minutes (numeric)
- delivery_time_minutes (numeric)
- delay_flag (0/1)
- delivery_delay_flag (0/1)
- analysis_completed_flag (0/1)

This schema is based on the hospital operations dataset.
